## Data Extraction Program
This project aims to extract names and phone numbers from unorganized data and put them into an organized file.

### Prerequisites

```bash
- Python 3.9 or later
- Python packages:
 - `re`
 - `csv`
 - `collections`
```

### How to Use
1. Clone this repository.
2. Install the required python packages.
